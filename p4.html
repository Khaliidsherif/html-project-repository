    1. OS Provides Graphical User Interface (GUI) in the form of menu, icons, and buttons.

    4. OS is responsible to manage the processes by making a process simple by threads.
    5. OS manage the memory by memory management techniques. e.g, paging, swapping, segmentation, etc.
    6. OS manage resource allocation. OS gives the resources to the process in such a way that all processes can use the resources in an efficient way.
